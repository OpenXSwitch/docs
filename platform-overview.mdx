### Platform Stack for Partners

|  |  |
| --- | --- |
| RESTful API | An OpenAPI 3.0 RESTful API designed to interact with the OpenXSwitch platform infrastructure, including wallets, programmable accounts, transactions, automation, and blockchain operations. |
| OpenXSwitch Console | A web-based operational dashboard for managing integrations, wallets, teams, API keys, transaction monitoring, compliance workflows, billing, and infrastructure configuration. |

### **How the platform is structured**

OpenXSwitch is designed as a modular platform composed of interoperable building blocks that work independently or together depending on your use case.

<Frame>
  ![Openxswitch Platform Section Nav](/images/openxswitch-platform-section-nav.png)
</Frame>

| Section | Description |
| :-- | :-- |
| **Accounts** | Organizational and administrative workspace for managing users, security, teams, webhooks, and API configuration. This is the default section users access after login. |
| **Wallet-as-a-Service** | Custodial wallet infrastructure for creating and managing wallets, sub-wallets, asset transfers, automation, treasury operations, and transaction monitoring across multiple blockchain networks. |
| **Smart Accounts (**`smart wallet`**)** | Programmable wallet infrastructure built on account abstraction principles, enabling gasless transactions, transaction sponsorship, stablecoin payments, auto-sweep infrastructure, and seamless onchain user experiences. |
| **Compliance Engine** | Compliance and monitoring tools for transaction risk analysis, wallet screening, address monitoring, alerts, and watchlist management. |

### Why OpenXSwitch?

|  |  |
| :-- | :-- |
| **Streamlined Infrastructure** | OpenXSwitch abstracts the technical, operational, and financial complexity of blockchain infrastructure, enabling businesses to build and scale digital asset products faster with lower operational overhead. |
| **Easy-to-Use APIs** | Developer-friendly APIs designed for rapid onboarding and scalable production usage, without requiring deep blockchain expertise or treasury management infrastructure. |
| **Frictionless User & Developer Experience** | Enable seamless stablecoin and wallet interactions without forcing users to manage gas fees, native tokens, seed phrases, or complex blockchain approval flows. |
| **Programmable Wallet Infrastructure** | Build advanced wallet experiences with transaction sponsorship, automation, programmable execution flows, and smart account orchestration. |
| **Built for Scale** | Designed for enterprises, fintechs, exchanges, payment platforms, and global applications operating across multiple blockchain ecosystems. |
